# `POST /auth/register`

- **接口中文名**：用户注册
- **请求体**：`RegisterRequest`

**Body 参数**

| 字段                  | 类型   | 必填 | 说明                                                        |
| --------------------- | ------ | ---- | ----------------------------------------------------------- |
| account               | string | 是   | 账号（学号/工号），为11位数字                               |
| name                  | string | 是   | 用户名                                                      |
| email                 | string | 是   | 邮箱                                                        |
| password              | string | 是   | 密码，至少8位，必须以英文字母开头，且只能包含英文字母和数字 |
| password_confirmation | string | 是   | 确认密码                                                    |
| code                  | string | 是   | 验证码，必须是6位                                           |

- **请求参数示例**

```json
{
    "account":"25010420551",
    "name":"伏明月1",
    "email": "3596976474@qq.com",
    "password": "fmy111111",
    "password_confirmation":"fmy111111",
    "code":"230699"
}
```

- **响应 `data`**：用户注册时信息

| 字段    | 类型   | 说明        |
| ------- | ------ | ----------- |
| id      | string | 注册用户 ID |
| account | string | 账号        |
| name    | number | 用户名      |
| email   | string | 邮箱        |
| role    | string | 角色        |

- **错误码**

| status | message                      | 条件               |
| ------ | ---------------------------- | ------------------ |
| 422    | 该账号已被注册，邮箱已被注册 | 账号和邮箱已被注册 |
| 400    | 验证码错误或已过期           | 验证码错误或过期   |

#### 8. 导出学生档案

**接口地址：** `POST /teacher/exports`

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| exportType | String | 是 | 导出类型：single(单个)/batch(批量)/all(全部) |
| studentIds | Array | 否 | 学生ID列表（batch时必填） |
| classId | Integer | 否 | 班级ID（all时可用） |
| exportFormat | String | 否 | 导出格式：pdf/excel，默认pdf |
| includeFields | Array | 否 | 包含字段列表，默认全部 |

**请求示例：**
```json
{
  "exportType": "batch",
  "studentIds": [1001, 1002, 1003],
  "exportFormat": "pdf",
  "includeFields": ["basicInfo", "academic", "awards", "practices"]
}
```

**响应示例：**
```json
{
  "code": 0,
  "message": "导出任务已创建",
  "data": {
    "taskId": 200,
    "status": 0,
    "statusLabel": "处理中",
    "estimatedTime": 30
  },
  "trace_id": "req-20260710-103000-abc123",
```
