### 新增文件

| 文件 | 说明 |
|------|------|
| service/user/CaptchaService.java | 图形验证码服务接口 |
| service/impl/user/CaptchaServiceImpl.java | 图形验证码服务实现（AWT 生成图片、内存存储、定时清理） |
| dto/response/user/CaptchaResponse.java | 验证码响应 DTO（key + base64 图片） |
| repository/log/LoginLogRepository.java | 登录日志仓库接口 |

### 修改文件

| 文件 | 说明 |
|------|------|
| controller/user/AuthController.java | 新增 `GET /auth/captcha` 端点，注入 CaptchaService |
| dto/request/user/LoginRequest.java | 新增 `captchaKey`、`captchaCode`、`rememberMe` 字段 |
| util/JwtUtil.java | 新增 `generateToken(userId, userNo, schoolId, expirationMs)` 重载方法 |
| service/impl/user/UserServiceImpl.java | 注入 CaptchaService/LoginLogRepository；新增验证码校验、记住密码逻辑；修复登录日志持久化 |
| config/SecurityConfig.java | 将 `/auth/captcha` 加入公开路径白名单 |
| config/security/JwtAuthenticationFilter.java | 将 `/auth/captcha` 加入放行路径列表 |

### 修改原因

1. **图形验证码**：登录页面应展示验证码图片，防止自动化暴力破解
2. **记住密码**：用户勾选"记住密码"时，JWT 有效期从 24 小时延长至 7 天，并在 users 表持久化 rememberToken
3. **登录日志持久化**：之前 recordLoginLog 方法仅创建对象但不入库，现注入 LoginLogRepository 完成真实写入
